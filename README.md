# DogList
Buscador de Imágenes de Perros por raza "Breed". Dog API con RecyclerView. Aplicación Android con Kotlin.

API utilizada: DOG API
-  Página Principal: [Dog CEO](https://dog.ceo/)
-  Página API: [Dog API](https://dog.ceo/dog-api/)
